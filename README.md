# Little Luxury Shop 💎

এটি একটি সুন্দর এবং সম্পূর্ণ কার্যকর অনলাইন শপ ওয়েবসাইট যা HTML, CSS এবং JavaScript দিয়ে তৈরি করা হয়েছে।

## বৈশিষ্ট্য ✨

- ✅ **সুন্দর ডিজাইন** - আধুনিক এবং রেসপন্সিভ লেআউট
- ✅ **পণ্য প্রদর্শন** - সমস্ত পণ্য একটি গ্রিডে দেখানো হয়
- ✅ **শপিং কার্ট** - পণ্য যোগ করুন, পরিমাণ পরিবর্তন করুন এবং অপসারণ করুন
- ✅ **অনুসন্ধান ফাংশন** - পণ্য নাম দিয়ে অনুসন্ধান করুন
- ✅ **ক্যাটাগরি ফিল্টার** - ক্যাটাগরি অনুযায়ী পণ্য ফিল্টার করুন
- ✅ **চেকআউট সিস্টেম** - অর্ডার সম্পূর্ণ করুন
- ✅ **মোবাইল বান্ধব** - সব ডিভাইসে নিখুঁতভাবে কাজ করে

## ফাইল স্ট্রাকচার 📁

```
little-luxury-shop/
├── index.html      # মূল HTML ফাইল
├── style.css       # CSS স্টাইলিং
├── script.js       # JavaScript কার্যকারিতা
└── README.md       # এই ফাইল
```

## কীভাবে ব্যবহার করবেন 🚀

### ১. ফাইলগুলি ডাউনলোড করুন

আপনার কম্পিউটারে একটি ফোল্ডার তৈরি করুন এবং সব ফাইল সংরক্ষণ করুন।

### ২. ওয়েবসাইট খুলুন

`index.html` ফাইলটি আপনার ব্রাউজারে খুলুন। যেকোনো ওয়েব ব্রাউজার কাজ করবে:
- Google Chrome
- Mozilla Firefox
- Safari
- Microsoft Edge

### ৩. কেনাকাটা শুরু করুন

- **পণ্য দেখুন**: সব পণ্য হোম পেজে দেখা যাবে
- **অনুসন্ধান করুন**: শীর্ষে খুঁজুন বক্সে পণ্যের নাম লিখুন
- **ফিল্টার করুন**: ক্যাটাগরি সিলেক্ট করে নির্দিষ্ট ধরনের পণ্য দেখুন
- **কার্টে যোগ করুন**: যেকোনো পণ্যের "কার্টে যোগ করুন" বোতাম ক্লিক করুন
- **কার্ট দেখুন**: শীর্ষে 🛒 কার্ট বোতাম ক্লিক করুন
- **পরিমাণ পরিবর্তন করুন**: কার্টে +/- বোতাম দিয়ে পরিমাণ বাড়ান বা কমান
- **চেকআউট করুন**: "চেকআউট করুন" বোতাম ক্লিক করুন

- **HTML5** - আধুনিক সেমান্টিক মার্কআপ
- **CSS3** - গ্রেডিয়েন্ট, ফ্লেক্সবক্স এবং গ্রিড লেআউট

- **LocalStorage** - কার্ট ডেটা সংরক্ষণ (ভবিষ্যতের সংস্করণে)

## কাস্টমাইজেশন গাইড 🎨

### রঙ পরিবর্তন করুন

`style.css` ফাইলে এই কালার কোডগুলি খুঁজে বের করুন এবং পরিবর্তন করুন:
- `#667eea` - প্রধান রঙ (নীল)
- `#764ba2` - গ্রেডিয়েন্ট রঙ (বেগুনি)
- `#ff6b6b` - একসেন্ট রঙ (লাল)

### পণ্য যোগ করুন

`script.js` ফাইলে `products` অ্যারেতে নতুন পণ্য যোগ করুন:

```javascript
{
    id: 9,
    name: 'আপনার পণ্যের নাম',
    category: 'watch', // বা jewelry, perfume, bag
    price: 10000,
    emoji: '💍',
    description: 'পণ্যের বর্ণনা'
}
```

### লোগো পরিবর্তন করুন

`index.html` ফাইলে এই লাইনটি খুঁজে বের করুন এবং পরিবর্তন করুন:

```html
<div class="logo">💎 Little Luxury Shop</div>
```
